# 🛡️ Guardian Health MVP

The Global Operating System for Traveler Care. This project is a functional prototype of the Guardian Health platform, featuring AI-driven triage and a premium, glassmorphic UI.

## 🚀 Deployment Guide

This project is built with **Vite + React** and is designed to be deployed with zero configuration on platforms like **Vercel**, **Netlify**, or **GitHub Pages**.

### 1. Deploy to Vercel (Recommended)
The easiest way to deploy is using the Vercel CLI or by connecting your GitHub repository.

**Via Vercel CLI:**
```bash
npm install -g vercel
vercel
```

### 2. Manual Build
To generate the production-ready static files:

```bash
cd guardian-mvp
npm install
npm run build
```
The output will be in the `dist/` directory, which can be served by any static web server.

## 🛠️ Tech Stack
- **Frontend**: React 18
- **Styling**: Vanilla CSS (Global Design System)
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Tooling**: Vite

## 📁 Project Structure
- `src/App.jsx`: Main application logic and UI.
- `src/index.css`: Global design system and glassmorphic styles.
- `index.html`: Entry point with optimized SEO meta tags.
